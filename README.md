# 📟 Linux Commands Database

Интерактивная база данных Linux-команд с подробными описаниями, флагами и примерами.

🔗 **Открыть:** `https://ВАШ_ЛОГИН.github.io/linux-cmds-db`

## Что внутри

- **100+ команд** по 13 категориям
- Для каждой команды: описание, все флаги, примеры с пояснениями, ссылки на man
- Поиск по командам, описаниям и флагам
- Работает офлайн (кроме внешних ссылок)
- Адаптирован для мобильных (iPhone/Android)

## Как развернуть

1. Создай репозиторий с именем `linux-cmds-db`
2. Загрузи файлы (`index.html`, `README.md`)
3. Перейди в **Settings → Pages**
4. Source: **Deploy from a branch** → ветка `main` → папка `/ (root)`
5. Нажми **Save** — через ~1 минуту сайт будет доступен

## Как открыть на iPhone

Просто открой ссылку в Safari — работает как веб-приложение.

Чтобы добавить на экран «Домой»:
1. Открой ссылку в Safari
2. Нажми кнопку «Поделиться» (квадрат со стрелкой)
3. «На экран Домой» → «Добавить»
